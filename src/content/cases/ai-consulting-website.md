## Problem

YouTube audience was growing but had nowhere coherent to land. Linktree looks amateur, and a full "agency" site would be premature. Needed a fast, credible landing focused on one primary CTA: book a call.

## Solution

A single Astro project with:

- **Hero + offer** — one sentence on what I do, one CTA
- **Methodology positioning** — AI data audits (the *how*), not industry verticals (the *who*) — because Russian construction experience ≠ US authority, so vertical claims would hurt credibility
- **Proof section** — YouTube + GitHub + case links
- **Deployed on Cloudflare Pages** via GitHub Actions

## Stack

Astro for static-first delivery (fast, cheap, clean), Tailwind for utility styling, TypeScript for component safety, Cloudflare Pages for global CDN on free tier.

## Why this matters

Positioning decisions > visual polish. The site exists because choosing methodology over vertical was the right call — supported by research, not vibes. That's the consulting product: pick the right cut, then execute.
